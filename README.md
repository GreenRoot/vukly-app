### Commits example

```gitexclude
- #24; feat: Adding new functionality.
- #24; fix: Fixing bugs.
- #24; refactor: Code changes that neither fix bugs nor add new features.
- #24; docs: Documentation changes.
- #24; style: Changes that do not affect the meaning of the code (e.g., formatting).
- #24; test: Adding or updating tests.
- #24; chore: Updates to the build process, configuration changes, etc.
```
